## Desafio Realizado

Neste desafio foram analisados os dados de quatro lojas utilizando gráficos e métricas de desempenho.  
Após a análise, a **Loja 1** foi escolhida como a melhor loja, pois apresentou o **maior faturamento** e a **segunda maior nota média** entre todas.
